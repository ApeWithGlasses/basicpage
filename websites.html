Sitios web

¿Qué es un sitio web?

Es una colección de páginas web que se agrupan y normalmente se conectan de varias maneras. Existen varios tipos de sitios web que entre ellos tenemos:

- De comercio electrónico.

- Landing page

- Institucionales

- Brand websites

- Blogs

- De servicios profesionales

- Foros, comunidades y redes sociales

- Wikis
