¿Qué es HTML?

HTML, lenguaje de marcas de hipertexto por sus siglas en inglés HyperText Markup Language es el componente más básico de una web. Este lenguaje de marcas de hipertexto define el significado y la estructura del contenido web, generalmente se utilizan otras tecnologías para describir la apariencia de una página web (CSS) o la funcionalidad (JS).
